# Contributing to ZamAI

We appreciate your interest in contributing to ZamAI projects!

## How to Contribute

1.  **Report Issues:** Use the GitHub Issues tab in the relevant repository to report bugs or request features.
2.  **Submit Pull Requests:** Fork the repository, create a branch, and submit a PR with your changes. Ensure your code follows the existing style and passes tests.

## Code Style

-   Follow standard conventions for the language (e.g., PEP 8 for Python, ESLint for JavaScript/TypeScript).
-   Write clear commit messages.
